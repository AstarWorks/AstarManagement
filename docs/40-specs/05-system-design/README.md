## 🎯 設計原則

1. **Clean Architecture** - レイヤー間の依存関係を明確化
2. **DRY (Don't Repeat Yourself)** - コードの重複を避ける
3. **SOLID原則** - 保守性と拡張性の高い設計
4. **KISS (Keep It Simple, Stupid)** - シンプルで理解しやすい実装
